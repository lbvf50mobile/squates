## Squates

Tool for logging training in CLI: `$ squates`  

* [Instalation](#instalation)  
* [Usage](#usage) 
* [TODO](#todo)

### Instalation
- `$ echo "export SQUATES_FILE=<file_for_log>" >> ~/.bashrc`
- `$ bash install.sh`

### Usage
- `$ allsquates` get log of last 25 rows from the file.
- `$ squates` save 10 squates row in the file file
